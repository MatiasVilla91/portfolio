[003] Temblor вҖ” Global 3D Seismograph в—Ҹ LIVE

Real-time global seismic visualizer built on CesiumJS: a 3D globe with real satellite imagery, high-performance WebGL rendering and native geographic coordinate support. It consumes up to 5,000 events from the USGS catalog and places them on the globe in real time. Real-time vector winds over 77 grid points from Open-Meteo. Configurable push alerts by magnitude. Dual seismic-risk model: an ETAS implementation in the browser (Aki b-value 1965, Omori-Utsu law, Gutenberg-Richter + Poisson) and an XGBoost model trained in Python on 2 years of catalog data, 10 seismic features and 5-fold cross-validation (AUC ~0.84вҖ“0.87). Fully client-side вҖ” no backend.

CesiumJS В· WebGL XGBoost В· ETAS Real-time В· USGS Open-Meteo Quantitative seismology No backend

[004] VoxCanis вҖ” Canine Vocalization Analysis в—Ҹ LIVE

Audio DSP + ML pipeline that extracts 67 acoustic features (pYIN, 13 MFCCs + deltas, HPSS, spectral centroid, onset detection), classifies the vocalization into 9 classes using a Random Forest trained with weak supervision over Freesound, and calls Claude Opus via SSE streaming to produce an interpretation grounded in peer-reviewed literature. Feature extraction in pure numpy/scipy in production to avoid OOM on free tiers.
